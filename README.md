# Minimal Kitchen (仮) 〜 在庫管理駆動型・お弁当レシピ提案システム 〜

「日々の献立考案にかかる思考コストをゼロにし、食材ロスをなくす。」

忙しい共働き世帯における「毎日のお弁当作り」という課題に対し、効率的かつ合理的なアプローチで解決を図るためのWebアプリケーションです。

開発者自身が前職の資材購買部門で10年間培った「在庫管理・納期管理」のロジックを、家庭の冷蔵庫に応用。「安売り食材」をシステムで的確に管理・消費することで、毎月の給食費相当（約400円/日）のコスト削減を実現しつつ、栄養バランスの取れた献立を自動で導き出します。

## コンセプトとターゲット

- **ターゲット:** 毎朝のお弁当作りに追われる、忙しい共働き世代
- **解決する課題:** 献立考案のストレス軽減、食材の廃棄ゼロ（SDGsへの貢献）、食費の削減
- **UI/UXのこだわり:** 忙しい朝でも視覚的なノイズにならない、余白を活かした「大人シンプル（Nadia風）」で洗練されたデザイン設計。

## コア機能

- **プロフェッショナル在庫管理（賞味期限アラート）**
    - 冷蔵庫内の食材をシステムで一元管理。設定した賞味期限に基づき、「今すぐ使い切るべき食材」をトップ画面で視覚的にアラート表示します。
- **お弁当カラーバランサー（彩り可視化ロジック）**
    - 各食材に「赤・黄・緑」の属性を紐付け。選択したレシピの彩りバランスが一目で確認でき、お弁当特有の「見栄えと栄養」の課題を直感的に解決します。
- **AI献立レコメンド（※実装予定）**
    - ユーザーの在庫データ（余り物食材）を元に、外部AI APIと連携。「子供が食べやすい」などの条件を加味した最適なレシピを自動提案します。
